## Requisitos para construir una custom rom

- CPU: 4 Núcleos o más
- RAM: 64 GB (también pueden ser 32 GB físicos y 32 GB de swap)
- Almacemiento: 500Gb SSD de espacio libre
- Sistema Operativo: Ubuntu 18.04 o mayor (distribuciones derivadas de Debian/Ubuntu o también puede ser Arch Linux)
- Git
- Repo
- Conocimientos básicos sobre el manejo de una terminal Linux

## Paquetes necesarios para construir Android

```sh title="Ubuntu 18.04 o mayor"
sudo apt-get install git-core gnupg flex bison build-essential zip curl zlib1g-dev libc6-dev-i386 x11proto-core-dev libx11-dev lib32z1-dev libgl1-mesa-dev libxml2-utils xsltproc unzip fontconfig
```

## Instalando y configurando 'repo'

<Steps>
1.  ```bash frame="none"
    curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo
    ```

2. ```bash frame="none"
    chmod a+x ~/bin/repo
    ```

3. ```bash frame="none"
    git config --global user.name 
    ```

4. ```bash frame="none"
    git config --global user.email
    ```
</Steps>
